[b]Добро пожаловать в Twitch Extended![/b]
Этот мод расширяет интеграцию Noita с Twitch, добавляя новые события, награды за баллы канала, награды за подписку и многое другое!

[Этот мод требует Config Lib, без него он не будет работать должным образом.]

[b]На данный момент этот мод добавляет:[/b]
[list]
    [*]61 новое событие
    [*]2 изменённых событий
    [*]3 переименованных событий
    [*]Изменена система выбора событий, увеличено время восстановления событий, чтобы одни и те же события не появлялись постоянно.
    [*]Голосование за снаряжение [Требуется Goki's Things]
    [*]Голосование за изменение забега 
    [*]Голосование за бонусы
    [*]Ники зрителей для чемпионов [Требуется Goki's Things]
    [*]Ники зрителей для мини-боссов [Требуется Goki's Things]
    [*]Награды за подписку
    [*]Награды за баллы канала
    [*]Меню для лёгкого подключения наград за баллы канала
    [*]Опция отключения системы голосования [по-прежнему позволяет использовать награды за баллы канала и т.п.]
    [*]Опция отображения чата в игре
    [*]Меню для удаления бонусов из голосования за бонус
    [*]Меню для отключения отдельных наград за подписку
	[*]Вы можете создавать свои собственные моды для добавления новых наград в этот мод, пример мода доступен.
[/list]

Подписаться на Goki's Things можно [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1975079109]здесь.[/url]

Если вы являетесь разработчиком и хотите добавить новые награды или модификаторы забега, на [url=https://github.com/EvaisaGiac/Twitch-Extended/releases]Github[/url] есть пример мода для их добавления.

Для доступа к новому меню настроек во время запуска нажмите кнопку Cog в правом верхнем углу экрана (она может перемещаться в зависимости от того, сколько модов добавляют новые кнопки).
